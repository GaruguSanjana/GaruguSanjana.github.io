HTML & CSS Notes

Simple notes for beginners

HTML

~HTML (Hypertext Markup Language) is the standard markup language for creating web pages.

~HTML elements are represented by tags, which indicate the structure of the page.

~Markup Language: HTML uses markup to annotate text, images,and other content to define its structure and meaning.

~Example HTML tag:

<h1>This tag is for font size</h1>
<table>It makes a table</table>
<p>This is to write a paragraph</p>
<ul>This is for unordered list</ul>
<ol>This is for ordered list</ol>
<a>Creates hyperlinks to other webpages</a>
<img>Embeds images to html document</img>
<li>Defines items in list</li>

CSS

~CSS (Cascading Style Sheets) is used for styling HTML elements on web pages.

~CSS allows you to control the layout, colors, fonts, and other visual aspects of your website.

~CSS selectors are patterns used to select and style HTML elements.

~CSS plays a crucial role in creating responsive web designs that adapt to different screen sizes and devices.

Example CSS rule:

h1 {
    color: blue;
    font-size: 24px;
}
